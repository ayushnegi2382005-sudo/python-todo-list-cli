# 📝 To-Do List (CLI)

A simple command-line To-Do List application built using Python.

## Features

- View tasks
- Add new tasks
- Delete existing tasks
- Saves tasks using a text file
- Input validation

## Technologies Used

- Python 3

## How to Run

```bash
python todo_list.py
```
